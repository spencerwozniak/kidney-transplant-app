# Kidney Transplant Navigation | Mobile App

Patient-facing mobile app that helps dialysis patients understand, track, and navigate the kidney transplant pathway using **patient-owned data**, **clear explanations**, and **actionable next steps**.

This app is designed to function independently of dialysis facilities and puts patients in control of their own health information.

---

## What This App Does (MVP)

- Presents a **patient journey timeline** showing where the patient is in the transplant pathway
- Allows patients to **upload and carry their own data** (labs, documents, questionnaires)
- Displays **plain-language explanations** of transplant eligibility factors
- Shows **explainable alerts** (rare, gated, non-alarming)
- Provides a **simple clinician-facing alert view** for transparency
- Emphasizes clarity, calm UX, and patient autonomy

---

## Core Screens

- Patient Journey Timeline (single source of truth)
- Data Upload & Checklist
- Eligibility Explanation Cards
- Consent Toggles (when to notify clinicians)
- Clinician Alert View (read-only, explainable)

---

## Tech Stack

- Expo React Native
- TypeScript
- REST API integration with PHP backend
- Mobile-first, patient-centered UX

---

## Design Principles

- Clarity over complexity  
- Patient ownership of data  
- Actionable guidance on every screen  
- Calm, explainable alerts (no alert fatigue)  

Design inspiration: Vessel-style healthcare UX (simple, empathetic, high-signal).
